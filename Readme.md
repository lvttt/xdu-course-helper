# XDU 选课助手（Chrome 扩展）

## 项目介绍
- 面向 `https://yjsxk.xidian.edu.cn/yjsxkapp/sys/xsxkapp/` 选课系统（覆盖登录页 `index.html` 与选课页 `course.html`）。
- 基于 Chrome Manifest V3 的 Side Panel 扩展：通过侧边栏提供开关与工具，同时在页面内注入脚本实现“课程收藏、批量选课”等功能。
- 后台 Service Worker 负责任务分发与网络请求，Content Script 在页面 MAIN world 内完成 XHR/组件 Hook 与 UI 增强。

## 功能速览
- **强制进入选课系统**：侧边栏一键进入选课页，并可在页面数据加载阶段注入必要信息以绕过部分时间限制（非选课时段请慎用）。
- **出错时自动重定向**：当接口返回异常、响应解析失败或出现 `NullPointer` 时，可自动清理 Cookie 并跳回登录页。
- **修改每页条数**：自定义选课列表每页显示数量（过大可能导致页面卡顿，建议不超过 100）。
- **课程收藏**：在课程列表增加“收藏/取消收藏”操作，并新增“已收藏课程”标签页管理收藏清单（数据保存在页面 localStorage）。
- **批量选课**：在“已收藏课程”中勾选课程后，可按顺序批量提交选课请求，支持配置两次请求间隔（默认 800ms，最小 200ms）。
- **循环批量选课**：对勾选课程按轮次循环尝试，可配置轮次间隔（默认 5000ms，最小 1000ms）；支持自动重新登录，连续失败达到阈值会自动停止。
- **记住账号密码 + 验证码辅助登录**：可保存账号密码用于后续登录，支持手动输入验证码登录与“验证码识别后自动登录”。
- **验证码识别服务**：内置 `vitphp`，支持配置多个自定义识别服务，组合“优先链路 + 回退链路”，并提供测试与导入导出。
- **一键清理**：侧边栏可清理 Cookie 并重新回到登录页；也可清除已保存的“记住账号”信息。
- **导出诊断信息**：导出扩展版本与配置摘要（已脱敏/不导出密码），便于排查问题。

## 直接安装方式
1) 获取构建产物（例如 GitHub Releases 中的压缩包），解压后确保目录内包含 `manifest.json`。
2) 打开 Chrome 扩展管理页 `chrome://extensions`，开启“开发者模式”。
3) 点击“加载已解压的扩展程序”，选择上一步的目录（通常是构建产物目录，例如 `dist/`）。
4) 打开目标站点页面后，点击扩展图标即可打开侧边栏；侧边栏会在非目标站点自动禁用。

## 使用指南
1) 访问 `https://yjsxk.xidian.edu.cn/yjsxkapp/sys/xsxkapp/index.html` 或 `course.html`，点击扩展图标打开侧边栏。
2) 常用侧边栏开关与工具：按需开启“强制进入选课系统 / 出错时自动重定向 / 修改每页条数”，并设置“批量选课请求间隔 / 循环批量选课间隔”。
3) 课程收藏与批量选课：在课程列表使用“收藏/取消收藏”维护清单，在新增的“已收藏课程”标签页勾选后执行“批量选课 / 循环批量选课”。批量选课的课程需要先在“培养计划”中选择，才能进行选班。
4) 记住账号密码与验证码登录：先在侧边栏开启“记住账号密码”，再在登录页完成一次正常登录以保存凭证。之后可在侧边栏“获取验证码”并手动输入登录，或配置“验证码识别服务”后使用“自动识别登录”；不再需要时可“清除记住账号信息”。
5) 遇到异常：可使用“清理 Cookie 并重新登录”快速回到登录页；可使用“导出诊断信息”生成 json 文件用于自查或反馈问题（不包含密码明文）。

## 注意事项
- 侧边栏仅在 `https://yjsxk.xidian.edu.cn/yjsxkapp/sys/xsxkapp/` 下启用，非目标站点会自动禁用。
- “记住账号密码”信息保存在页面 localStorage 中（由选课站点域名隔离，但仍属于本机可读数据），不建议在公共设备上使用；必要时及时在侧边栏清除。
- 验证码识别内置服务会访问第三方接口（例如 `vitphp`），可能存在稳定性、限流与隐私风险；建议按需配置自定义服务并使用回退链路。
- 自定义验证码服务需要访问外部域名时，浏览器可能会提示授予对应域名的访问权限；请确认接口可信后再授权。
- 批量/循环批量选课属于自动化行为，可能触发站点风控；建议设置合理间隔并自行承担风险。
