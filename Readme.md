# XDU 选课助手（Chrome 扩展）

## 项目简介
- 基于 Vue 3 + Vite 的浏览器扩展，面向 `https://yjsxk.xidian.edu.cn/yjsxkapp/sys/xsxkapp/` 选课系统。
- 通过侧边栏提供快捷操作：强制进入选课页、错误重定向、分页条数调整、记住账号密码、验证码获取与自动登录。
- 后台脚本在 `src/background` 处理任务分发和配置，内容脚本在 `src/content` 注入页面逻辑，UI 组件在 `src/ui`。

## 功能速览
- **强制进入选课系统**：跳过时间限制，一键打开选课界面（非选课时段请慎用）。
- **出错自动重定向**：检测到常见异常（如 NullPointer）时清理 Cookie 并回到登录页。
- **修改每页条数**：自定义课程列表分页数量（建议不超过 100 条）。
- **记住账号密码**：加密存储凭证，支持手动/自动识别验证码后登录。
- **一键清理**：遇到白屏或异常时，可清理 Cookie 并重新登录。

## 安装与构建
- 环境：Node.js 18+，推荐使用 npm。
- 安装依赖：`npm install`
- 构建发布包：`npm run build`（产物在 `dist/`）
- 直接安装：前往最新[Release](https://github.com/lvttt/xdu-course-helper/releases/latest)下载 zip 压缩包，解压后在 Chrome 扩展页加载解压目录；部分浏览器也支持直接选择 zip 文件加载。

## 使用指南
1) 打开 Chrome `chrome://extensions`，开启开发者模式，选择“加载已解压的扩展程序”，指向 `dist/`。
   - 若使用 Release 的 zip 包，解压后同样选择该目录；若浏览器支持，可直接选 zip 文件。
2) 访问 `https://yjsxk.xidian.edu.cn/yjsxkapp/sys/xsxkapp/`，浏览器会自动启用侧边栏入口。
3) 在侧边栏按需开启开关：需要强制进入时点击“立即进入选课系统”；遇到错误/白屏时使用“清理 Cookie 并重新登录”；调整分页或记住账号后按提示获取验证码并登录。

## 开发命令
- `npm run dev`：启动 Vite 开发服务，便于调试 UI。
- `npm run build`：生产构建输出到 `dist/`。
- `npm run preview`：预览构建产物。
- `npm run format`：使用 Prettier 格式化主要源码文件。

## 注意事项
- 仅在目标域名下会显示侧边栏功能，其他站点会自动禁用。
- 记住密码功能仅存储加密后的凭证，不建议在公共设备上使用。
- 使用强制进入与分页调整时，留意学校的选课开放时间与页面性能。
